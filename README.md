# Nyaa Preview

Automatically loads and displays the largest images from external links in nyaa.si torrent descriptions.

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/)
2. Install the [Nyaa Preview](https://raw.githubusercontent.com/love98ooo/nyaa-preview/main/nyaa-preview.user.js) script.

## Usage

Visit any `https://sukebei.nyaa.si/view/*` page.

For the first time, you need to allow cross-origin requests.

![Allow Cross-Origin Requests](./Always%20allow%20all%20domains.jpg)

The script will automatically fetch and display images from external links in the torrent description.
